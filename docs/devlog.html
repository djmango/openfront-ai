Bugs (hall of fame)

  • Win detection (Jul 6 12:06) - username vs clientID. Fixed.
  • Phantom boats (Jul 7) - legal mask but silent no-op + no penalty. v4.2 fix.
  • Resume vs supervisor - stale --stage fought checkpoint. Checkpoint authoritative now.
  • Case-insensitive macOS - map dir lookup broke on Linux.
  • Cropped-map index crash - boat coords off trimmed grid edge.
  • Auto-restart without backoff - OOM crash loops for hours; commit assert added.

Lessons

  • Only compress what's big; exact small state bypasses the latent.
  • Make wins reachable before making them valuable (1v1 staging).
  • Warm starts inherit stale habits - retrain when reward/curriculum changes materially.
  • An action that silently fails at noop cost is a free lottery ticket - mask it or price it.
  • Benchmark on the distribution you'll deploy on (bot AE dropped 16pt on human borders).
  • Spatial precision comes from resolution, not channels.
  • Watch the agent play - curves missed win bug and phantom boats.
  • Pad to the batch max, not the curriculum max.
  • Treat training hosts as disposable; checkpoint + HF sync everything.
  • Sequence models multiply memory before they multiply quality.
