bc2 throughput decay - encode is the feed cost

Throughput decayed 100 → 25 ex/s over ~8 hours; a restart fully fixed it. Phase logging (smp/enc/col) showed the Rust sampler at ~0s, collate ~5-9s, and frozen-AE encode at 50-135s per window - the encode shares the GPU with the trainer. CUDA reserved flat at 27.5GB, so not fragmentation or thermal throttle. bf16 autocast on the AE encode brought bc2 back to ~85-93 ex/s. Decay recurred within 30 minutes; suspect glibc arena churn from numpy H2D staging. Running MALLOC_ARENA_MAX=2; durable fix is ofrs returning pre-stacked batches.

July 6th, 2026 - late night

Rust port and the phantom-boat fix, ~midnight through 11 p.m.

v4.2 - phantom boat exploit fixed mid-run

Replay sidecars: 43-78% of decisions were boat, nearly all silently discarded (max 3 transports). Discarded intents paid like noop with occasional upside - free lottery tickets.

Fixed resume-safe (no tensor shape changes): honest masks, valid-tile snapping, W_WASTE=0.01 per discarded intent. Hot-deployed to ppo_v4 at update 260; first v4.2 episodes measured ~85-93 wasted intents per episode.

Rust round two - full BC sampler and binary vec IPC

DataLoader process workers regressed - 16 workers idle while the main process choked unpickling ~50MB batches. Replaced with ofrs::Sampler (decode + featurize + labels in Rust/rayon, 639 → 2631 samples/s laptop, exact parity on 11.8k samples), binary vec.py IPC off pickle, and native collate. Pod results: bc2 14.9 → ~105 ex/s; bc3 seq 4.5 → ~10.5 ex/s.

Earlier same stretch: initial ofrs decode+collate; free-threaded CPython 3.14t verified locally (BC sampler 1.7× at 4 threads).

Win detection was broken

vec.py checked winner[1] == "Agent"; engine emits clientID "AGENTRL1". Every win-rate stat before noon is partly invalid. Found by watching a replay where the agent visibly won while stats said otherwise.

Reference

Reward (v5)

rl/curriculum.py / rl/vec.py. Dense: 0.02 × strength × timeweight (0.5→1.0 by tick 8000). Delta: gain 5.0 / loss 6.5. Terminal: 15 × rank-1.5 + 30 win bonus. Death −1, wasted intent −0.01. Strength: 45% land / 25% mil / 30% eco.

Curriculum

11 stages, 7 maps. Win-gated advancement, map pools, 25% rehearsal. Early 1v1 → 1v3 so wins are reachable before they matter.

Lessons

  • Only compress what's big; exact small state bypasses the latent.
  • Make wins reachable before making them valuable.
  • Warm starts inherit stale habits when reward/curriculum changes.
  • A silently-failing "legal" action is a free lottery ticket - mask it or price it.
  • Benchmark on the distribution you'll deploy on.
  • Watch the agent play; curves missed the win bug and phantom boats.
